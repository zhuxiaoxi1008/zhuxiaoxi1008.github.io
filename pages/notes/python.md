
### python 笔记


``` python
    # 更新 pip
    python -m pip install --upgrade pip

    # python -m pip install --upgrade package_name
    # python -m pip install --user package_name

    pip install pytest

    ```
    文件名称 用 test_ 开头
    第一，测试函数必须以 test_ 打头。在测试过程中，pytest 将找出并运行所有以 test_ 打头的函数。
    第二，测试函数的名称应该比典型的函数名更长，更具描述性。
    ```
    
    python -m pytest

```
